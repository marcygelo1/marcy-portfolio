# Project Description
This project is a personal portfolio showcasing various works and skills.

# Setup Instructions
1. Clone the repository: `git clone https://github.com/marcygelo1/marcy-portfolio`
2. Navigate to the project directory: `cd marcy-portfolio`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

# Technologies Used

- CSS
- JavaScript
- React
- Node.js
